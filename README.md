## 下载发布版本 (推荐的)

1. 下载 [最新的发布版本](https://www.pagekit.com/download)。
2. 解压并且复制到服务器目录。
3. 在浏览器中打开目录 i.e. `http://localhost/pagekit` 然后跟随安装程序。

## 开发版本安装

如果你想安装现在的开发版本, 你可以 [开发版本安装](https://pagekit.com/docs/developer/source).

## CLI

Pagekit 提供一系列的命令去运行单独的任务。详细命令如下
```
./pagekit --help
```
